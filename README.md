# Генетик Хавф Бахолаш — Streamlit версия

Қисқа тавсиф:
- Тил: Узбекча интерфейс
- Мақсад: Хомиладор аёлларда ирсий касалликлар хавфини тез баҳолаш (демонстрация)

Қандай жойлаштириш (Streamlit Cloud):
1. Ушбу кодни GitHub репозиторига push қилинг (файллар: `app.py`, `requirements.txt` ва бошқа ёрдамчи файллар).
2. https://share.streamlit.io га киринг ва "New app" → репо танланг → branch ва `app.py` файлини ёзилганлигини текширинг.
3. Deploy тугмасини босинг — бир неча дақиқа ичида саҳифа ишга тушади.

Мухим:
- Ҳар доим тиббий тавсия олиш учун малакали мутахассисга мурожаат қилинг. Ушбу дастур клиник тасдиқга эга эмас.
- Агар реал генетик модел ёки тренировка қилинган ML моделингиз бўлса, ўша модельни `compute_risk_score` функцияси ўрнига юклаб ишлатинг (joblib, pickle ва ҳ.к.).
